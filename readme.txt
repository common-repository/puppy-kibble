Puppy Kibble Widget for WordPress

== Description ==
Puppy Kibble is a simple widget that shows a random puppy image from Instagram in a widget area on every page load. All of these images have been vetted myself.

== Installation ==
1. Activate the plugin through the admin panel on your site.
2. Goto Appearance -> Widgets and Drag the Puppy Kibble Widget to a Sidebar Area.
